Table Structure
Elements Definition
Table The <table>tag defines an HTML table.
tr The <tr>tag defines a row in an HTML table.
th The <th>tag defines a header cell in an HTML table.
td The <td>tag defines a standard data cell in an HTML table.
headers The <thead>tag is used to group header content in an HTML table.
body The <tbody>tag is used to group the body content in an HTML table.
footers The <tfoot>tag is used to group footer content in an HTML table.
captions The <caption>tag defines a table caption.
colspan To make a cell span over multiple columns, use the colspan attribute.
rowspan To make a cell span over multiple rows, use the rowspan attribute.
